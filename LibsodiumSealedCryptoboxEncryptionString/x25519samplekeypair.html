curve X25519 sample keypair

The curve X25519 is used for key exchange mechanisms or short KEM. Each party has it's own key pair so below you will find 2 pairs. Usual names for these parties are "Alice" and "Bob" but I will shorten it to "A" and "B" key pair. Each key is only 32 bytes long and for better (cross platform wide) usage encoded in Base64 so the string is a little bit longer.
